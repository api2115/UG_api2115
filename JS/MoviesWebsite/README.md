# MoviesWebsite
Prosta biblioteka filmów łącząca się z API. Do jej stworzenia użyty został ReactJS.

## Funkcjonalność:
- Strona główna ze wszystkimi dostępnymi filmami.
- Sortowanie filmów wg. kryteriów.
- Widok szczegółowy dla każdego dostępnego filmu.
- Widok z filmami dodanymi jako ulubione.
- Dodawanie filmów.
- Usuwanie filmow.
- Edycja w widoku szczegółowym.
